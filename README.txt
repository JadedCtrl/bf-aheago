===============================================================================
BRAINFUCK AHEAGO
===============================================================================

A simple lisp implementation of Brainfuck.
Currently can work with simpler programs; but for now, it's still sorta
fragile.

> (bf-aheago:interpret #p"examples/hello_world.bf" :tape-length 5)
